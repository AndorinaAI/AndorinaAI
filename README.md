## 👤 About

I build practical, open-source tools using AI as the main development engine. My role is to come up with the idea, design the architecture, write the specifications, test every edge case by hand, debug what breaks, and ship it.

I don't write most of the code myself — I use AI to generate it. What I *do* bring is the judgment: knowing when something is right, when it's wrong, and why.

> Building tools I actually want to use, on Linux, with zero telemetry.

## 🚀 Featured Project

<div align="center">

[![Andoriña](https://github-readme-stats.vercel.app/api/pin/?username=AndorinaAI&repo=Andorina-WhatsApp-Agent-for-Hermes&theme=midnight-purple&hide_border=true&bg_color=0d1117&title_color=a855f7&text_color=c9d1d9&icon_color=a855f7)](https://github.com/AndorinaAI/Andorina-WhatsApp-Agent-for-Hermes)

</div>

**Andoriña** — an autonomous WhatsApp manager that runs as a skill on top of [Hermes Agent](https://github.com/andorinaAI/Andorina-WhatsApp-Agent-for-Hermes). It turns your AI into a full messaging autopilot: schedule messages, send files and voice notes, manage contacts, set semantic keyword alerts, and control it all from a local web panel.

- 🔒 100% local processing. No cloud, no telemetry.
- 🛡️ Zero-trust security pipeline with RBAC, input guards, and DLP.
- 🔄 One-click GitHub updater that preserves all your data.
- 🐧 Linux only (by choice).
