# Hi there! 👋

I'm the creator of Andoriña. I come from a WordPress background, and now I focus on designing, structuring, and thoroughly testing software while using AI to write the code.

### How I build
I don't write most of the code by hand. My job is to:
1. Come up with the idea and design the architecture.
2. Write clear specifications.
3. Let AI generate the implementation.
4. Manually test and debug every single edge case until it works perfectly.

I build tools I actually want to use, with a strong focus on running locally (Linux) and having zero telemetry.

### Featured Project: Andoriña
An autonomous WhatsApp manager that runs as a skill on top of [Hermes Agent](https://github.com/AndorinaAI/Andorina-WhatsApp-Agent-for-Hermes). It turns your AI into a messaging autopilot: schedule messages, send files and voice notes, manage contacts, set semantic keyword alerts, and control it all from a local web panel.

👉 [Check out Andoriña here](https://github.com/AndorinaAI/Andorina-WhatsApp-Agent-for-Hermes)
